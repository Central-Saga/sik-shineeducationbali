  PANDUAN CLEAR CACHE BRAVE UNTUK shine.local.test
═══════════════════════════════════════════════════════════

LANGKAH 1: HAPUS HSTS
────────────────────────────────────────────────────────────
1. Buka: brave://net-internals/#hsts
2. Scroll ke "Delete domain security policies"
3. Ketik: shine.local.test
4. Klik "Delete"

LANGKAH 2: CLEAR CACHE
────────────────────────────────────────────────────────────
SHORTCUT: Cmd+Shift+Delete (Mac) atau Ctrl+Shift+Delete (Windows)
- Time range: All time
- Centang: Cached images and files + Cookies
- Klik "Clear data"

ATAU via Settings:
1. Menu ≡ > Settings > Privacy and security
2. Clear browsing data > Advanced tab
3. Time range: All time
4. Centang: Cached images + Cookies
5. Clear data

LANGKAH 3: RESTART BRAVE
────────────────────────────────────────────────────────────
1. Tutup SEMUA window (Cmd+Q)
2. Buka kembali Brave
3. Akses: https://shine.local.test
